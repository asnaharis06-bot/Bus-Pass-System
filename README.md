# 🚌 BusPass — Cloud-Based Bus Pass System
CodeAlpha Cloud Computing Internship | Task 2

A fully functional cloud-based bus ticket booking system built with HTML, CSS, JavaScript and Firebase. Supports secure login, real-time booking, and digital QR tickets.

## ✨ Features
- 🔐 Secure user authentication (Login & Signup)
- 🎫 Digital QR-coded tickets stored in the cloud
- 🚌 Multiple bus routes and pass types
- 📈 Auto-scaling infrastructure using Firebase
- 💻 Real-time database with Firestore
- 📱 Fully responsive on mobile and desktop
- 🛡️ Anti-theft tickets tied to user account

## 🛠️ Tech Stack

| Tool | Purpose |
|------|---------|
| `HTML/CSS/JavaScript` | Frontend UI |
| `Firebase Authentication` | User login & signup |
| `Firebase Firestore` | Cloud database for tickets |
| `Firebase Hosting` | Live cloud deployment |
